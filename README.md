# WER Measurement Project

This project aims to measure the Word Error Rate (WER) for various Automatic Speech Recognition (ASR) and Text-to-Speech (TTS) services, including real-time OpenAI.

## Project Structure

- `asr/`: Contains scripts and configurations for ASR services.
- `tts/`: Contains scripts and configurations for TTS services.
- `scripts/`: Contains scripts for running WER measurements and analyzing results.
- `config.json`: Configuration file for specifying service credentials and other settings.
- `README.md`: This file.
